# 🧮 Calculator Project

This is a modern and responsive Calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean user interface and interactive features.

## 🚀 Features
- Basic arithmetic operations (+, −, ×, ÷)
- Responsive and user-friendly design
- Interactive buttons with hover gradient effects
- Real-time input and result display
- Clear (C) and delete (⌫) functionality
- Keyboard support (Enter, Backspace, numbers)
- Smooth animations and styling

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 How to Use
1. Open the project folder
2. Double-click `index.html`
3. Use buttons or keyboard to perform calculations

## 📌 Project Purpose
This project is created to practice and demonstrate basic front-end development skills including UI design, event handling, and JavaScript logic.
